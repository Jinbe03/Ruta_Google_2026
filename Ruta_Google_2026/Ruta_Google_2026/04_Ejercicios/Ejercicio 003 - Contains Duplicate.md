## Sintaxis Clave

- for numero in nums: Bucle simple para revisar cada elemento.
    
- if numero in memoria: La pregunta fundamental de existencia.
    
- return True / return False: El manejo de la salida dependiendo de si el return esta dentro o fuera del bloque for.
    

## Flujo del Codigo

Creamos un diccionario vacio al inicio. Iniciamos un bucle for. Dentro, usamos un if para ver si el numero ya es una llave en nuestra memoria. Si el if es positivo, salimos con return True. Si el bucle termina (identacion hacia la izquierda), significa que el if nunca fue cierto, por lo que ejecutamos return False.
