# Comparativa de Estrategias

## 1. Diccionario (Manual)

- Mas control sobre el proceso.
    
- Util para cuando necesitas hacer algo extra al encontrar el duplicado.
    
- Mas propenso a errores de indentacion.
    

## 2. Set (Optimizado)

- Codigo mucho mas corto: return len(nums) != len(set(nums)).
    
- Python hace el trabajo pesado internamente.
    
- Es la forma mas "Pythonica" de resolverlo.
