# Comando: range()

Sirve para: Crear una secuencia de números de forma automática. Se usa casi siempre junto al bucle for.

## Como funciona

Es una fábrica de números. Si le pides un range(5), te entrega los números del 0 al 4.

## Sintaxis basica

for i in range(5): # i tomara los valores 0, 1, 2, 3, 4

## Ejemplo real

for paso in range(1, 4): print(Paso numero, paso)
