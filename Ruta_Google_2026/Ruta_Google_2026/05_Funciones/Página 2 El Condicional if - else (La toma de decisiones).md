# Comando: if / else

Sirve para: Hacer que el programa tome caminos distintos dependiendo de una condiciÃ³n (Verdadero o Falso).

## Como funciona

Es como un guardia de seguridad. Si cumples la regla (if), pasas por una puerta. Si no la cumples (else), pasas por otra o te quedas fuera.

## Sintaxis basica

if condicion: # Se ejecuta si es Verdad else: # Se ejecuta si es Falso

## Ejemplo real

edad = 20 if edad >= 18: return Puedes pasar else: return Eres menor
