# Comando: in

Sirve para: Preguntar si un elemento existe dentro de un grupo (lista, diccionario o texto).

## Como funciona

Es como buscar tu nombre en una lista de invitados. En lugar de leer toda la lista tú mismo, el comando in lo hace por ti y te dice Sí o No.

## Sintaxis basica

if elemento in grupo: # Hacer algo si existe

## Ejemplo real

frutas = [pera, manzana] if manzana in frutas: print(Si hay manzanas)
