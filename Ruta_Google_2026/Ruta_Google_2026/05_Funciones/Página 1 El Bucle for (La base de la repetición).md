# Comando: for

Sirve para: Ejecutar un bloque de código muchas veces, una por cada elemento que haya en una lista o colección.

## Como funciona

Imagina que tienes una fila de personas. El for toma a la primera persona, hace algo con ella, luego toma a la segunda, y así hasta llegar al final.

## Sintaxis basica

for elemento in lista: # Codigo que se repite

## Ejemplo real

numeros = [1, 2, 3] for n in numeros: print(n * 2)
